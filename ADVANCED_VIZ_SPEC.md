# Advanced Population Dynamics Visualization Specification

> **Document Type**: Frontend Implementation Guide
> **Created**: 2026-01-12
> **Purpose**: Specifications for interactive visualization of advanced population dynamics models

---

## Overview

This document specifies the **interactive visualizations** needed to explore the advanced population dynamics analysis, including:

1. **Noise Decomposition Dashboard** - Process vs observation uncertainty
2. **Population Growth Rate Explorer** - Interactive λ estimates with uncertainty
3. **Survival Probability Surfaces** - Size-dependent vital rates by genus
4. **State-Space Model Viewer** - Time series with hidden states
5. **IPM Visualization** - Matrix heatmaps and eigenvalue analysis
6. **Uncertainty Quantification Panel** - Credible intervals and predictions

---

## Data Contracts

### Input JSON Files

All data files are generated by `scripts/R/06_advanced_population_models.R` and located in `public/data/`:

#### 1. `survival_predictions.json`

```typescript
interface SurvivalPrediction {
  genus: "Poc" | "Por" | "Acr" | "Mil";
  size: number;              // Colony size in cm
  survival_prob: number;     // Predicted survival probability [0, 1]
  survival_lower: number;    // 95% CI lower bound
  survival_upper: number;    // 95% CI upper bound
}

type SurvivalData = SurvivalPrediction[];
```

**Usage**: Plot size-dependent survival curves with credible intervals.

#### 2. `population_dynamics_summary.json`

```typescript
interface PopulationDynamicsSummary {
  lambda_estimates: {
    genus: string;
    lambda: number;          // Population growth rate
    log_lambda: number;      // Log scale
    status: "Declining" | "Stable" | "Increasing";
  }[];

  noise_decomposition: {
    genus: string;
    median_process_sd: number;     // Process noise standard deviation
    median_obs_sd: number;         // Observation error SD
    q025_process: number;          // 2.5% quantile
    q975_process: number;          // 97.5% quantile
    q025_obs: number;
    q975_obs: number;
    ratio: number;                 // Process/Observation ratio
    dominant: "Process" | "Observation" | "Mixed";
  }[];

  timestamp: string;
}
```

**Usage**: Summary statistics for dashboard overview.

#### 3. `uncertainty_quantification.json`

```typescript
interface UncertaintyQuantification {
  genus: string;
  lambda_median: number;
  lambda_mean: number;
  lambda_sd: number;
  lambda_q025: number;           // 95% CI bounds
  lambda_q975: number;
  prob_declining: number;        // P(λ < 1)
  prob_stable: number;           // P(1 ≤ λ ≤ 1.05)
  prob_increasing: number;       // P(λ > 1.05)
}

type UncertaintyData = UncertaintyQuantification[];
```

**Usage**: Visualize posterior distributions and risk assessments.

#### 4. `population_time_series.json`

```typescript
interface PopulationTimeSeries {
  genus: string;
  genus_full: string;
  year: number;
  N: number;                     // Abundance
  mean_size: number;             // Mean colony size
  recruitment: number;           // Number of recruits
  pop_growth_rate: number;       // log(N_t / N_{t-1})
}

type TimeSeriesData = PopulationTimeSeries[];
```

**Usage**: Interactive time series charts with multiple series.

#### 5. `measurement_error.json`

```typescript
interface MeasurementError {
  measurement_sd: number;        // Standard deviation in cm
  measurement_cv: number;        // Coefficient of variation
}
```

**Usage**: Display measurement precision context.

```typescript
interface GrowthRateExplorerProps {
  lambdaData: UncertaintyData;
  onSelectGenus?: (genus: string) => void;
}

const PopulationGrowthExplorer: React.FC<GrowthRateExplorerProps> = ({
  lambdaData,
  onSelectGenus
}) => {
  const [selectedGenus, setSelectedGenus] = useState<string | null>(null);

  const handleClick = (genus: string) => {
    setSelectedGenus(genus);
    onSelectGenus?.(genus);
  };

  return (
    <div className="space-y-4">
      {/* Main chart: point + error bars */}
      <svg width={800} height={400}>
        {lambdaData.map((d, i) => {
          const x = 100 + i * 150;
          const y = yScale(d.lambda_median);
          const yLower = yScale(d.lambda_q025);
          const yUpper = yScale(d.lambda_q975);

          return (
            <g key={d.genus} onClick={() => handleClick(d.genus)}>
              {/* Error bar */}
              <line x1={x} y1={yLower} x2={x} y2={yUpper}
                    stroke="#333" strokeWidth={2} />
              <line x1={x-10} y1={yLower} x2={x+10} y2={yLower} stroke="#333" />
              <line x1={x-10} y1={yUpper} x2={x+10} y2={yUpper} stroke="#333" />

              {/* Point estimate */}
              <circle cx={x} cy={y} r={8}
                      fill={getStatusColor(d)}
                      stroke="#fff" strokeWidth={2}
                      className="cursor-pointer hover:r-12" />

              <text x={x} y={height - 10} textAnchor="middle">
                {d.genus}
              </text>
            </g>
          );
        })}

        {/* Reference line at λ = 1 */}
        <line x1={0} y1={yScale(1)} x2={width} y2={yScale(1)}
              stroke="red" strokeDasharray="5,5" />
      </svg>

      {/* Detail panel for selected genus */}
      {selectedGenus && (
        <DetailPanel data={lambdaData.find(d => d.genus === selectedGenus)!} />
      )}
    </div>
  );
};
```

```typescript
interface SurvivalSurfaceProps {
  data: SurvivalData;
  selectedSize?: number;
  onSizeChange?: (size: number) => void;
}

const SurvivalProbabilitySurface: React.FC<SurvivalSurfaceProps> = ({
  data,
  selectedSize = 10,
  onSizeChange
}) => {
  // Group by genus
  const dataByGenus = useMemo(() =>
    groupBy(data, d => d.genus), [data]
  );

  // D3 scales
  const xScale = d3.scaleLog()
    .domain([1, 100])
    .range([0, width]);

  const yScale = d3.scaleLinear()
    .domain([0, 1])
    .range([height, 0]);

  return (
    <div className="space-y-4">
      <svg width={800} height={500}>
        {/* Plot each genus */}
        {Object.entries(dataByGenus).map(([genus, points]) => {
          const color = genusColorMap[genus];

          // Line path
          const line = d3.line<SurvivalPrediction>()
            .x(d => xScale(d.size))
            .y(d => yScale(d.survival_prob))
            .curve(d3.curveBasis);

          // Confidence interval area
          const area = d3.area<SurvivalPrediction>()
            .x(d => xScale(d.size))
            .y0(d => yScale(d.survival_lower))
            .y1(d => yScale(d.survival_upper))
            .curve(d3.curveBasis);

          return (
            <g key={genus}>
              {/* CI ribbon */}
              <path d={area(points) || ""}
                    fill={color} opacity={0.2} />

              {/* Mean line */}
              <path d={line(points) || ""}
                    stroke={color} strokeWidth={3} fill="none" />
            </g>
          );
        })}

        {/* Vertical line at selected size */}
        <line x1={xScale(selectedSize)} y1={0}
              x2={xScale(selectedSize)} y2={height}
              stroke="#999" strokeDasharray="3,3" />

        <Axes xScale={xScale} yScale={yScale} />
      </svg>

      {/* Size slider */}
      <RangeSlider
        min={1}
        max={100}
        value={selectedSize}
        onChange={onSizeChange}
        scale="log"
        label="Select Size (cm)"
      />

      {/* Predictions at selected size */}
      <PredictionTable
        data={data.filter(d =>
          Math.abs(d.size - selectedSize) < 0.5
        )}
      />
    </div>
  );
};
```

#### Interactions

- **Slider**: Update vertical line and table
- **Hover curve**: Show tooltip with exact survival probability
- **Click genus legend**: Toggle visibility
- **Right-click**: Context menu to export CSV of predictions

```typescript
interface StateSpaceViewerProps {
  timeSeriesData: TimeSeriesData;
  noiseData: PopulationDynamicsSummary["noise_decomposition"];
}

const StateSpaceModelViewer: React.FC<StateSpaceViewerProps> = ({
  timeSeriesData,
  noiseData
}) => {
  const [selectedGenus, setSelectedGenus] = useState("Poc");
  const [showObserved, setShowObserved] = useState(true);
  const [showLatent, setShowLatent] = useState(true);

  const genusData = timeSeriesData.filter(d => d.genus === selectedGenus);
  const genusNoise = noiseData.find(d => d.genus === selectedGenus);

  // Estimate latent state (simplified - actual would use Kalman filter)
  const latentState = genusData.map(d => ({
    year: d.year,
    N_latent: d.N,  // Placeholder
    N_lower: d.N - 1.96 * genusNoise!.median_process_sd * d.N,
    N_upper: d.N + 1.96 * genusNoise!.median_process_sd * d.N
  }));

  return (
    <div className="space-y-4">
      {/* Controls */}
      <div className="flex gap-4">
        <Select value={selectedGenus} onChange={setSelectedGenus}>
          {["Poc", "Por", "Acr", "Mil"].map(g => (
            <option key={g} value={g}>{genusFullName(g)}</option>
          ))}
        </Select>

        <Checkbox checked={showObserved} onChange={setShowObserved}>
          Observed
        </Checkbox>
        <Checkbox checked={showLatent} onChange={setShowLatent}>
          Latent State
        </Checkbox>
      </div>

      {/* Time series plot */}
      <svg width={800} height={400}>
        {/* Latent state with CI */}
        {showLatent && (
          <g>
            <path d={areaGenerator(latentState)} fill="#999" opacity={0.3} />
            <path d={lineGenerator(latentState)} stroke="#333" strokeWidth={2} />
          </g>
        )}

        {/* Observed points with error bars */}
        {showObserved && genusData.map((d, i) => {
          const x = xScale(d.year);
          const y = yScale(d.N);
          const error = genusNoise!.median_obs_sd * d.N;

          return (
            <g key={i}>
              <line x1={x} y1={yScale(d.N - error)}
                    x2={x} y2={yScale(d.N + error)}
                    stroke="#666" strokeWidth={1} />
              <circle cx={x} cy={y} r={5} fill="#4daf4a" />
            </g>
          );
        })}
      </svg>

      {/* Summary stats */}
      <div className="grid grid-cols-2 gap-4">
        <StatCard label="Process SD" value={genusNoise?.median_process_sd} />
        <StatCard label="Observation SD" value={genusNoise?.median_obs_sd} />
      </div>
    </div>
  );
};
```

### Visualization Library Choice

**Option 1: D3.js** (Recommended for advanced/custom visualizations)
- ✅ Full control over SVG rendering
- ✅ Best for novel visualization types
- ✅ Excellent performance
- ❌ Steeper learning curve

**Option 2: Plotly** (Recommended for standard scientific plots)
- ✅ Built-in statistical charts
- ✅ Interactive by default
- ✅ Easy to implement
- ❌ Less customization

**Option 3: Recharts** (For rapid prototyping)
- ✅ React-native API
- ✅ Simple declarative syntax
- ❌ Limited advanced features

**Recommendation**: Use **D3 + Recharts hybrid**:
- D3 for Survival Surface, IPM Matrix
- Recharts for time series, bar charts
